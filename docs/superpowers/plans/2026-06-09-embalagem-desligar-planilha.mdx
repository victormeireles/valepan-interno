# Embalagem — Desligar planilha (Fase C) — Plano de implementação

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Meta e realizado de embalagem passam a gravar e ler **somente Supabase**; planilha `Pedido de embalagem` congelada; sync desligado.

**Architecture:** Strangler por endpoint — substituir dual-write + `reconcileForDate` por serviços DB-only (`OrdemProducaoMetaService`, refatoração de `EmbalagemLoteService`). `congelado` vem de `tipos_estoque`; remove `etiqueta_gerada`. Rollout: Pré → Meta → Realizado → Limpeza.

**Tech Stack:** Next.js App Router, Supabase, Vitest, TypeScript.

**Spec:** `docs/superpowers/specs/2026-06-09-embalagem-desligar-planilha-design.mdx`
